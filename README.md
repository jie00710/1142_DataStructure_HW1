# 1142_DataStructure_HW1

## 使用方式

這個程式是一個具備基本算術、邏輯判斷與位元運算功能的終端機計算機。

### 啟動方式

1. 在專案根目錄執行：

```bash
g++ HW1.cpp -o HW1
./HW1
```

2. 程式啟動後會出現選單：

- `1`：基本算術與邏輯判斷
- `2`：位元運算
- `3`：離開程式

### 模式 1：基本算術與邏輯判斷

在此模式中可以輸入中序（infix）表示式，程式會自動轉成後序（postfix）並計算結果。

#### 支援的運算子

- `+`：加法
- `-`：減法
- `*`：乘法
- `/`：除法
- `(`、`)`：括號
- `<`：小於
- `>`：大於
- `<=`：小於等於
- `>=`：大於等於
- `==`：等於
- `!=`：不等於
- `AND`：且
- `OR`：或
- `XOR`：互斥或
- `NOT`：非

#### 範例

```
2 + 3 * 4
3 < 5 AND 4 == 4
3 >= 5
3 <= 5
3 == 3
3 != 5
```

輸入 `/0` 可以返回主選單。

### 模式 2：位元運算

在此模式中可以進行二進位相關運算，程式會保留位元寬度並以二進位格式輸出結果。

#### 支援的運算

- `|`：或
- `&`：與
- `^`：互斥或
- `~`：補數
- `<<`：左移
- `>>`：右移

#### 輸入格式

- 可以直接輸入十進位數字，例如：`10`
- 也可以輸入二進位字面值，例如：`0b1010`
- 也支援不帶前綴的二進位表示，例如：`1010`（會被當成二進位值處理）

#### 範例

```
0b1010 | 0b0011
~0b1111
0b1000 << 2
0b1000 >> 1
1010>>
1010 AND1100
```

#### 注意事項

- 在位元運算模式中，`AND`、`OR`、`XOR`、`NOT` 也可以直接使用關鍵字來表示。
- 位元運算結果會以固定位元寬度顯示，例如 `1010` 右移後仍會保留格式。

### 退出程式

選擇主選單中的 `3` 即可離開程式。
